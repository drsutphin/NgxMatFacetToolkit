# v1.0.0 Release Notes

## Highlights
- Standalone-only Angular 19 facet toolkit with Material UI.
- New package name and selector: `@drsutphin/ngx-mat-facet-toolkit`, `ngx-mat-facet-toolkit`.
- Updated provider configuration via `provideFacetToolkitConfig`.
- Expanded documentation with API reference, theming guidance, and facet cookbook.

## Breaking Changes
- `NgxMatFacetSearchModule` removed (standalone-only).
- Selector changed from `ngx-mat-facet-search` to `ngx-mat-facet-toolkit`.
- Package name changed to `@drsutphin/ngx-mat-facet-toolkit`.

## Upgrade Notes
- Replace old imports with `@drsutphin/ngx-mat-facet-toolkit`.
- Update templates to use the new selector.
- Move any module-level config into `provideFacetToolkitConfig` during bootstrap.
- Include the toolkit theme mixins in your global theme.

## Verification
- `npm run test`
- `npm run build`
